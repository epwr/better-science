package log // import "github.com/epwr/better-science/internal/common/eplog" A package that provides basic structured logging. TYPES type CallerLocation struct { Function string File string Line int } A location in the code. Should not be used externally to this package. func get_caller_location() CallerLocation Get the location (function name, file, and line number) of where this function's caller was called from (2 levels back down the call stack). type LogRecord struct { Level string Message string Caller CallerLocation } A record that stores the core information for logging purposes.
